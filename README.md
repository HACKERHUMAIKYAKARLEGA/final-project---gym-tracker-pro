# 🏋️ GymTracker Pro

A modern, responsive, and feature-rich **Multi-Gym Management System** built using **HTML, CSS, and JavaScript**. GymTracker Pro helps gym owners, managers, and trainers efficiently manage members, trainers, memberships, and workout records through an intuitive dashboard.

---

## ✨ Features

### 🔐 Authentication
- Login & Sign Up system
- Role-based registration
- Secure authentication interface
- Owner, Manager, and Trainer roles

### 🏢 Multi-Gym Management
- Register multiple gyms
- Manage each gym independently
- Gym-specific users and data

### 👥 Member Management
- Add, edit, and delete members
- Search and filter members
- Assign trainers
- Membership plans:
  - Basic
  - Standard
  - Premium
- Membership status:
  - Active
  - Expired
  - Suspended

### 🏋 Trainer Management
- Add and manage trainers
- Trainer profiles
- Assign members to trainers
- Trainer-specific dashboard

### 📋 Workout Management
- Log workouts
- Track exercise type
- Duration tracking
- Calories burned
- Workout history
- Notes for each workout

### 📊 Dashboard
- Total Members
- Total Trainers
- Total Workouts
- Average Workouts per Member
- Monthly Workout Chart
- Membership Statistics
- Recent Activity
- Quick Actions

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile
- iPhone/iPad optimized
- Touch-friendly interface
- Bottom navigation for mobile

### 🎨 Modern UI
- Dark theme
- Smooth animations
- Interactive cards
- Toast notifications
- Responsive tables
- Modal forms
- Beautiful typography
